
## Countdown

Start a countdown. This command only updates every 10 seconds to avoid being ratelimited by Discord

Usage:

```md
!countdown|cd [hours=0] [minutes=0] [seconds=0] [title]
```

Aliases:

```md
cd
```

Examples:

```md
None
```
