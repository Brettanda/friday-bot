# Music



## Stop



Usage:

```none
stop|disconnect 
```

Aliases:

```none
!disconnect
```

## Play

Follow this command with the title of a song to search for it or just paste the Youtube/SoundCloud url if the search gives and undesirable result

Usage:

```none
play|p|add <url/title>
```

Aliases:

```none
!p,!add
```
