# Music



## Custom

Play sounds/songs without looking for the url everytime

Usage:

```md
!custom|c <name>
!custom|c remove <name>
!custom|c list 
!custom|c change <name> <url>
!custom|c add <name> <url>
```

Aliases:

```md
c
```

Examples:

```md
None
```
